📌 Acerca de esta Aplicación

Esta aplicación fue desarrollada con Angular para la gestión eficiente de tareas.

🔍 ¿Qué hace esta aplicación?

Permite visualizar tareas pendientes y completadas, además de consultar información del usuario responsable de cada tarea.

🔗 API Utilizada

Los datos de esta aplicación se obtienen desde JSONPlaceholder, un servicio gratuito para simular datos de una API.
